Browsers compatible with LightWAVE

The LightWAVE client runs in a web browser and does not require any platform-specific components. The browser must support HTML5, JavaScript, and (for graphical output only) SVG. The table below shows the oldest LightWAVE-compatible version of each browser.

[1] Pan-and-zoom, introduced in LightWAVE 0.19, is not fully functional on Linux in Firefox 9 (bug 543965) but it works in Firefox 15 and later (and probably in some earlier versions also).

[2] Although MSIE 6, 7, and 8 do not support SVG, you can use them to run LightWAVE if you install Google Chrome Frame (a free plugin that provides the missing support).
