About LightWAVE

LightWAVE 0.01

LightWAVE is a lightweight waveform and annotation viewer and editor.

LightWAVE is modelled on WAVE, an X11/XView application I wrote and maintained between 1989 and 2012. LightWAVE runs within any modern web browser and does not require installation on the user's computer.

This is an early alpha version of LightWAVE, which I wrote as a testbed and proof-of-concept. It is barely useful, providing a way to select and extract data in text form from PhysioBank records (with output similar to the ATM's 'show signals as text' and 'show annotations as text' tools). At this time, LightWAVE does not provide any graphical output related to the data selected by the user (the grid and cartoonish "waveforms" are placeholders for the not-yet-implemented interactive display).

It is likely that almost all of the current version will be revised before the initial public release, which is planned for January 2013.
