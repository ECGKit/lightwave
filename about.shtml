About LightWAVE

LightWAVE 0.03

LightWAVE is a lightweight waveform and annotation viewer and editor.

LightWAVE is modelled on WAVE, an X11/XView application I wrote and maintained between 1989 and 2012. LightWAVE runs within any modern web browser and does not require installation on the user's computer.

This is an early alpha version of LightWAVE, which I wrote as a testbed and proof-of-concept. It is barely useful, providing a way to select and extract data in text form from PhysioBank records (with output similar to the ATM's 'show signals as text' and 'show annotations as text' tools). At this time, LightWAVE does not provide any graphical output related to the data selected by the user (the grid and cartoonish "waveforms" are placeholders for the not-yet-implemented interactive display).

LightWAVE's back end is a nearly finished CGI application written in C for speed and efficiency. It delivers data in response to requests generated by the front end, and it is usable now; the current version includes some test code (notably the retrieve function, which fetch replaces) that will be removed before the first release. The back end can also be run as an interactive or scripted command-line application for testing, debugging, and possibly other purposes I haven't dreamed up just yet. The API is designed to ease development of alternative front ends with the aim of encouraging experimentation.

LightWAVE's front end will be substantially revised before the initial public release, which is planned for January 2013. It currently includes a GUI for selecting data structured as in PhysioBank (with DBS, RECORDS, and ANNOTATORS text indices and WFDB-compatible binary data files). Optimum buzzword compliance, browser independence, and general cross-platform goodness, is achieved via HTML5, CSS, and JavaScript/jQuery-generated Ajax requests for JSON-formatted data transformed into SVG elements that are dynamically injected into the DOM.

LightWAVE's documentation is currently limited to comments embedded in its source files (especially its Makefile) and this file.

LightWAVE is available for experimentation on PhysioNet at http://physionet.org/lightwave/ (live version); sources are available in the LightWAVE project on PhysioNetWorks. At this time, if you wish to develop alternative front ends, you will need to set up the back end and the Apache web server on your own machine. This requirement will be removed before the first public release.
